## Overview

The Gurduwara Langar Game is an interactive educational application designed to teach users about Sikh culture and the tradition of Langar (the community kitchen) through an engaging gamified experience. Developed as an impactful employee resource group (ERG) project at PayPal, this application serves as a cultural bridge, making the principles of selfless service, community, and equality accessible to a global audience. By simulating the process of preparing and serving in a Langar, the game offers a unique window into one of the most cherished aspects of Sikh heritage.

> This project was conceived to foster greater understanding and appreciation of Sikh traditions within the PayPal community and beyond, leveraging technology to create an immersive and educational cultural experience.

## Key Features

*   **Interactive Gamification:** Engages users with game mechanics that simulate the experience of participating in Langar service.
*   **Cultural Education:** Provides valuable insights into the history, philosophy, and practice of Langar and Sikhism.
*   **Cross-Platform Availability:** Built with React Native, the application is designed to run seamlessly on both iOS and Android devices.
*   **Community Impact:** Developed as a community-driven project to promote cultural diversity and inclusion.

## Architecture and How It Works

The application is built on a modular architecture using React Native, which allows for a single codebase to be deployed on both iOS and Android platforms. The core of the application is its game engine, which manages the state of the Langar simulation. This includes tracking user progress, managing virtual ingredients, and simulating the cooking and serving processes. Educational content is integrated throughout the game, with informational pop-ups and narrative elements that explain the cultural significance of the actions being performed.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and the React Native development environment set up.

*   [React Native Environment Setup](https://reactnative.dev/docs/environment-setup)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Ripnrip/GurduwaraLangarGame.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run on iOS or Android
    ```sh
    npx react-native run-ios
    # or
    npx react-native run-android
    ```
