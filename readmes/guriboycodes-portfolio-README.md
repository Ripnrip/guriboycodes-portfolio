## Overview

This repository contains the source code for my personal portfolio website, live at [guriboycodes.com](https://guriboycodes.com). This project serves as a comprehensive showcase of my career as a Staff Software Engineer, my journey through 28+ hackathons, and my passion for leveraging technology to create impactful solutions. The website is designed to impress and inform, providing a deep dive into my technical skills, project experience, and community involvement. It is built with a modern, responsive tech stack to ensure a seamless experience across all devices.

> This portfolio is more than just a collection of projects; it's a narrative of my growth as an engineer and a testament to my dedication to the craft of software development. It is designed to be a living document, constantly evolving as I take on new challenges and explore new technologies. The primary audience for this portfolio includes hiring managers and engineering leaders at top-tier technology companies, and the design and content are tailored to meet their expectations for a world-class software engineering candidate.

## Key Features

*   **Ghibli-Style AI Art:** Each project is accompanied by a unique, AI-generated illustration in the style of Studio Ghibli, created using the Google Nano Banana model via the Replicate API. This adds a touch of creative flair and visual storytelling to the portfolio, making it stand out from traditional text-based project lists. This feature demonstrates a creative application of generative AI to enhance user experience.

*   **Interactive Career Timeline:** A visually engaging and interactive timeline that details my career progression, key achievements, and the impact I've had on over 90 million users. This provides a clear and concise overview of my professional experience, highlighting my growth and contributions at major tech companies.

*   **Hackathon Journey:** A dedicated section that chronicles my participation in over 28 hackathons, highlighting my 2 first-place wins and showcasing my ability to innovate under pressure. This section demonstrates my passion for rapid prototyping and creative problem-solving, as well as my ability to collaborate effectively in a team environment.

*   **Functional Contact Form:** A fully functional contact form that allows visitors to get in touch with me directly, facilitating communication and networking opportunities. The form is implemented with a serverless function to handle submissions securely and reliably.

*   **Responsive Design:** The website is fully responsive, ensuring a great user experience on desktops, tablets, and mobile devices. This is achieved through the use of Tailwind CSS and a mobile-first design approach, ensuring that the content is accessible and easy to navigate on any screen size.

## Architecture/How It Works

The portfolio is a single-page application built with React and Vite, providing a fast and responsive user experience. Styling is handled by Tailwind CSS, with components from the shadcn/ui library for a clean and modern look. The AI-generated images are created by calling the Replicate API, which runs the Google Nano Banana model to generate images based on project descriptions. The website is deployed on Vercel for continuous integration and delivery, ensuring that the latest changes are always live.

The application state is managed using React's built-in state management features, such as the `useState` and `useContext` hooks. This keeps the application lightweight and avoids the need for external state management libraries. The project structure is organized by feature, with each component and its related files located in the same directory. This makes the codebase easy to navigate and maintain.

The architecture is designed to be scalable and maintainable. The use of a component-based architecture allows for easy reuse of UI elements, and the separation of concerns between the UI, business logic, and data fetching layers makes the codebase easy to reason about and extend. The use of Vercel for deployment simplifies the process of getting the application into the hands of users, and the platform's built-in features for continuous integration and delivery ensure that the application is always up-to-date.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine. You will also need a `.env` file with the following environment variables:

```
REPLICATE_API_TOKEN=your_replicate_api_token
```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ripnrip/guriboycodes-portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
